# I18n Sync Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace all hardcoded Vietnamese text with `$t()` calls so every string in the UI responds to the language switcher.

**Architecture:** The app already has vue-i18n v9 installed with 3 locale files (en/vi/zh) and a working `LanguageSwitcher` in the navbar. We only need to (1) add missing keys to the 3 locale files, then (2) replace each hardcoded string in templates/scripts with the corresponding `$t()` call.

**Tech Stack:** Vue 3, vue-i18n v9 (Composition API, `legacy: false`), Vite
